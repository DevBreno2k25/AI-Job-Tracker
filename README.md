# AI-Job-Tracker


A clean and responsive front-end web application built with HTML, CSS and Vanilla JavaScript to help users track job applications.

This project focuses on real-world logic, clean UI and simple, readable code — without frameworks.



 Features

- Add job applications (title, company, status)
- Delete applications
- Persistent data using **LocalStorage**
- Responsive layout (desktop & mobile)
- Modern dark UI (dashboard style)
- Smooth card animations
- Clean and beginner-friendly JavaScript logic



  Technologies Used

- HTML5  
- CSS3 (Flexbox & Grid)  
- Vanilla JavaScript  
- Browser LocalStorage  

No frameworks. No libraries.
